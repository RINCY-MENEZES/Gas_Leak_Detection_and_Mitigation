# 🔥 Intelligent IoT-Based Gas Leak Detection and Automated Safety System

An IoT-based gas leak detection and automated safety system developed using **ESP32-C3**, **MQ-6 Gas Sensor**, **SIM800L GSM Module**, and multiple safety mechanisms. The system continuously monitors LPG gas concentration and automatically performs safety actions during gas leakage.

---

## 📌 Features

- Real-time LPG gas leak detection
- Automatic buzzer and alerts
- SMS notifications using GSM module
- Automatic exhaust fan activation
- Automatic gas supply shut-off using solenoid valve
- Multi-layer safety response system
- Continuous gas concentration monitoring

## ⚙️ Working Principle

The MQ-6 gas sensor continuously monitors LPG gas concentration. When the gas concentration exceeds the predefined threshold, the ESP32-C3 immediately activates the buzzer and LED indicators, sends an SMS alert using the GSM module, switches ON the exhaust fan, and closes the gas supply using a solenoid valve. This multi-layer response minimizes fire hazards and enhances user safety.
